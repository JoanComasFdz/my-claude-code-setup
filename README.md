# my-claude-code-setup

A personal collection of the tools and configs I use when working with
[Claude Code](https://docs.claude.com/en/docs/claude-code).

It's meant to grow over time — right now the main piece is a ready-to-use
dev container.

## Run Claude Code in an isolated box

The safest way to let Claude Code work freely is to run it inside an isolated
container instead of directly on your machine

To use it in your repo:
1. Copy the `.devcontainer/` folder into that repo
2. Open your repo in VS Code.
3. A popup will appear: Reopen in Container.

See [`.devcontainer/README.md`](.devcontainer/README.md) for the full details.
